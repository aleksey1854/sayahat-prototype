# Саяхат — мультилендинги базара (прототип)

Кликабельный прототип платформы для рынка Саяхат, Костанай: каталог магазинов
базара и конструктор лендингов, которым может пользоваться даже непродвинутый
продавец.

## Страницы
- **index.html** — главная базара: каталог магазинов, поиск, фильтр по
  категориям, новости, карта рядов.
- **shop.html** — пример лендинга арендатора (магазин «Дастархан»).
- **constructor.html** — конструктор лендинга: пошаговый мастер, готовые
  шаблоны, загрузка фото (drag-drop / с телефона), живой предпросмотр, SEO.

## Технологии
Статический сайт — HTML / CSS / JS, без сборки. Открывается напрямую в браузере
или раздаётся через GitHub Pages. Шрифты Onest + Manrope.
